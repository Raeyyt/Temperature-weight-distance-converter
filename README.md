# Premium Unit Converter

A sleek and responsive web application for converting everyday units. Redesigned with a focus on modern UI/UX, featuring a custom dark green palette and glassmorphism effects.

## Key Features
*   **Instant Conversion:** Real-time calculation for Temperature (Celsius/Fahrenheit), Weight (Kg/Lbs), and Distance (Km/Miles).
*   **Modern UI:** Glassmorphic cards with a premium dark gradient background.
*   **Responsive:** Fully optimized for desktop and mobile devices.
*   **Lightweight:** Built with pure HTML, CSS, and JavaScript (no heavy frameworks).

## Technologies Used
*   HTML5
*   CSS3 (Variables, Flexbox, Glassmorphism)
*   JavaScript (ES6)
*   Google Fonts (Outfit, Inter)

## How to Run
1.  Clone the repository.
2.  Open `index.html` in your browser.
